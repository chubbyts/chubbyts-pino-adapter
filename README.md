## Description

Pino adapter for chubbyts-log-types, [PSR-3][2] inspired.

## Requirements

 * node: 18
 * [@chubbyts/chubbyts-log-types][3]: ^2.0.1
 * [pino][4]: ^9.6.0

## Installation

Through [NPM](https://www.npmjs.com) as [@chubbyts/chubbyts-pino-adapter][1].

```ts
npm i @chubbyts/chubbyts-pino-adapter@^2.0.0
```
